CouchPotatoTV

CouchPotatoTV is a Couch Potato client for Apple TV.

Installation

This installation assumes a few things:

  • You already have Couch Potato installed — CouchPotatoTV is just a way to interact with Couch Potato and not a downloader.
  • You're confortable with setting up a local PHP server.
  • You have a free Apple Developer account

There are two components required for it to work: the client app and the server. Both of these require some confirguration before installing them.

Server

The server is a PHP and Javascript app that returns TVML templates to the Apple TV.

Download .zip Clone on GitHub

There are example files included in the source (/config.example.php and /js/Config.example.js). Copy these example files and rename them /config.php and /js/Config.js respectively and update the variables for your setup.

  • $clientDomain - The domain or IP address of server itself, e.g. http://ip-address/cptv/
  • $CPDomain - The domain or IP addresss of your Couch Potato installation, usually http://ip-address:5050
  • $CPApiKey - Your Couch Potato API key, found in settings (show advanced)

Note: You cannot use localhost as the IP address, because the Apple TV client will not be running on the same machine as Couch Potato or the server. Get the IP address of the machine Couch Potato and the server are running from your computer's network settings.

  1. Download and unzip (or clone from GitHub) to wherever your local server is serving files from.
  2. Navigate to yourserver/wanted.php in your browser and you should see output similar to the screenshot below.

tvOS Client

The client is a Swift application that connects to the server to retrieve the templates.

Download .zip Clone on GitHub

There is only one configuration file for the client: AppDelegate.swift. Change the line static let TVBaseURL = "http://localhost:9001/" to point to the server IP address you setup in the previous steps.

To install the app to your Apple TV use the following steps:

  1. Connect your Apple TV to your Mac
  2. Open the project in Xcode
  3. Change the build identifier name to something unique (e.g. add your name to the end of the name).
  4. In the drop-down box in the toolbar, select your Apple TV
  5. Press the "play" button to run the app.
  6. You may be asked to select a development login for provisioning. Choose your Apple developer account for the dropdown.
  7. Once installed, you can quit Xcode and disconnect your Apple TV.
